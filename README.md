# THIS DOES NOT COME WITH YOSEMITE
# YOU MUST PROVIDE YOUR OWN COPY OF `YOS.DGO`

- BEFORE installing the mod
  - Make sure you have a copy of `YOS.DGO` - mount the September 2001 Preview ISO and copy it from the `DGO` folder
  - Go to the Jak 1 launcher page, and click Advanced -> Open Game Data Folder
  - Go into `iso_data/jak1/DGO`, paste your copy of `YOS.DGO` here
- Install mod
- Have fun
